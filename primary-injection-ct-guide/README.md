# Primary Injection CT Field and Placement Education Guide

Standalone HTML educational demo for primary injection breaker test set output-field visualization and monitoring CT placement.

## How to run

Open `index.html` in any modern browser. No build tools, dependencies, server, or internet connection are required.

## What it demonstrates

- Series shunt reference current
- Monitoring CT placement on a shorting cable
- Upper and lower output terminal geometry
- Magnetic field visualization around the high-current cable
- Placement quality scoring
- CT uncertainty contributors such as centering, jaw closure, burden, knee voltage, cabinet steel influence, leakage flux, thermal drift, and core nonlinearity

## Important limitation

This is an educational quasi-static conductor-field model. It is not finite-element analysis, certified calibration software, or a substitute for measured data from the actual CT, shunt, burden, and test set geometry.
